================================================================================

Validation Query:
SELECT COUNT(DISTINCT fkkvkp.zzacctmngr) AS bp_count
FROM fkkvkp
WHERE fkkvkp.zzacctmngr IS NOT NULL
  AND TRIM(fkkvkp.zzacctmngr) <> ''

================================================================================
RULE 14: ZSTAFF BP (NOT IN FKKVKP)
================================================================================

Program Logic: BUT000 + BUT0ID(type='ZSTAFF') + ZISCSALLOW(inactive<>'X')

FROM but000 b
INNER JOIN but0id i ON b.partner = i.partner
INNER JOIN ziscsallow z ON i.idnumber = z.emp_id
WHERE z.inactive <> 'X'
  AND z.parent_id <> ' '
  AND i.type = 'ZSTAFF'

================================================================================
RULE 15: GROUP BILL MASTER ACCOUNTS
================================================================================

WHERE fkkvk.vktyp = '02'

================================================================================
RULE 16: ENLIGHT CONVERSION FLAGGED BP/CA
================================================================================
Description: BP/CA flagged with short_text = 'ENLIGHTDATACONVERSION'
Program Logic: ZISCSBREMARK + ZISCSCAREMARK

Validation Query - BP:
SELECT COUNT(DISTINCT bp) AS bp_count
FROM ziscsbpremark
WHERE short_text = 'ENLIGHTDATACONVERSION'

Validation Query - CA:
SELECT COUNT(DISTINCT ca) AS ca_count
FROM ziscscaremark
WHERE short_text = 'ENLIGHTDATACONVERSION'

================================================================================
RULE 17: PERSON-PERSON RELATIONSHIPS
================================================================================
Description: All relationships between Group Bill master and child accounts
Program Logic: BUT050.PARTNER2 NE '' AND RELTYP NE 'ZDUP'

Validation Query:
SELECT COUNT(DISTINCT partner2) AS bp_count
FROM but050
WHERE partner2 IS NOT NULL
  AND TRIM(partner2) <> ''
  AND reltyp <> 'ZDUP'
